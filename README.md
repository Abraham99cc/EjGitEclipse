1. Primero me he creado una cuenta en github para luego poder crear un repositorio remoto vacio
2. En eclipse he abierto una nueva perspectiva git desde el boton de la parte superior derecha "open perspective".
3. Desde la ventana de la parte izquierda le he dado a clonar repositorio, en la parte de la url a clonar he puesto el enlace facilitado, despues he cambiado la url por la de mi repositorio remoto.
4.He importado el proyecto a mi explorador y alli he creado un nuevo fichero llamado README.md, que es este mismo.
5. Desde la ventana Git Staging he pasado el fichero README.md desde Unstaged Changes (1) a Staged Changes.
6.Finalmente pondré en el Commit Message mi nombre y apellidos para luego hacer commit para confirmar los cambios.
7. Después push para subir los cambios a mi repositorio remoto.
